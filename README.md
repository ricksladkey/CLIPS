The following site is the upstream repository:

* [clipsrules.sourceforge.net](http://sourceforge.net/p/clipsrules/code/HEAD/tree/)

Features:

* Visual Studio 2012 solutions and projects
* Ported to and bug-fixed for x64/AMD64 64-bit platform
